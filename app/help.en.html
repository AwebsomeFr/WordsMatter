Installation

This documentation deals exclusively with the use of the WordsMatter Application. To learn how to install and configure the WordsMatter Application and API, see the post "Créer un blog écoresponsable avec WordsMatter" (fr) published by Awebsome or the technical README of the GitHub repository (fr).

User interface

The user interface of the WordsMatter Application is divided into two areas.

The upper part of the screen is the preview area, the lower part is the writing area.
Write from the writing space; the preview area delivers, in real time, a preview of the generated layout.

The main menu is accessible from the upper right corner of the screen with the Menu button.

In the opposite corner, the connexion indicator specifies whether or not the Application was able to connect to its API at launch.

Main menu

Save (CTRL + S)

Save the on-screen publication to the web browser's local memory.

There can only be one post saved in this way.

The post will be restored when opening the WordsMatter Application.

Web Storage API required.

Delete

Delete the saved post (on screen or not) from the local memory of the web browser.

Web Storage API required.

Push to the blog (CTRL + P)

Start sending the on-screen post for posting or saving as a draft on the blog.

WordsMatter API required.

Edit from the blog (CTRL + O)

List blog posts and drafts for editing or deletion.

WordsMatter API required.

Export to .txt file

Provides the raw content of the screen post for manual saving to a .txt file.

Import from .txt file

Reloads on screen a post from a .txt file created with the previous option.

The content on the screen will be overwritten by the imported one.

Manage gallery (CTRL + G)

Allows adding and removing images from the gallery.

WordsMatter API required.

Light theme / dark theme

Switch between light theme and dark theme.

Adjust width

Set the occupied screen width.

Requires a screen width of at least 768px.

Adjust height

Set the distribution of the writing and preview spaces in height.

Syntax

WordsMatter uses some Markdown but brings its own subtleties.
The examples below detail the syntax to respect and show the default rendering specific to each formatting.

Title of the post

Fill in the text field whose label is "Title of the post (h1)".

Required for posting on the blog.

Section title

Fill in the text field(s) whose label is "Title of the section (h2)".

Section subtitle

Buttons "h3" / "h4" / "h5" / "h6", or text to transform, at the beginning of the line, immediately preceded by the desired title level and an underscore.

3_title h3
4_title h4
5_title h5
6_title h6

title h3

title h4

title h5
title h6

New paragraph

Press the [Enter] key once.

Line break

Type two spaces at the end of the line ...
... before pressing the [Enter] key once.

Strong text

"Str" button, or text to transform immediately preceded and followed by two underscores.

__strong text__

strong text

Emphasic text

"Em" button, or text to transform immediately preceded and followed by an underscore.

_emphasic text_

emphasic text

Ordered list

Button "ol" (where each new line acts as delimiter), or text to transform preceded by two underscores and a space at the beginning of the line.

__ element n°1
__ element n°2
__ ...

  1. element n°1
  2. element n°2
  3. ...

Unordered list

Button "ul" (where each new line acts as delimiter), or text to transform preceded by an underscore and a space at the beginning of the line.

_ element n°1
_ element n°2
_ ...

  • element n°1
  • element n°2
  • ...

Hyperlink

Button "a", or text to transform, first in brackets for the label, then immediately in parentheses for the link.

[Label of the link](https://targeted-domain.com)

Label of the link

Image

"Img" button, or text to transform preceded by an exclamation point, alternative of the image between brackets, link between brackets.

![WordsMatter logo](./img/words-matter-logo-100px.png)

Image with legend

Button "fig", or text to transform preceded by an exclamation point, alternative and caption of the image between brackets and delimited by a vertical separator, link between parentheses.

![WordsMatter logo|The WordsMatter's logo](./img/words-matter-logo-100px.png)

Underscore

The underscore (_) is considered special. Preceded by a backslash, it can however be used as such.

\_name\_of\_file

name_of_file

Otherwise, it can be interpreted and give rise to unexpected results.

\_name_of_file

nameoffile
